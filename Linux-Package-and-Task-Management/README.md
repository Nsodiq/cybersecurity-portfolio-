# Linux Package and Task Management

## Objective
Learn the fundamentals of Linux package management, repositories, package verification, and scheduled task concepts.

## Skills Learned
- Linux package management
- Software repository management
- Package verification concepts
- Scheduled task management
- Linux administration fundamentals

## Concepts Studied

### Package Management
- apt
- apt remove
- add-apt-repository
- dpkg

### Package Verification
- GPG

### Scheduled Tasks
- Crontabs
- MIN
- HOUR
- DOM
- MON
- DOW
- CMD

## What I Learned
I learned how Linux systems install and manage software packages, how repositories are used to distribute software, and how cron jobs automate recurring tasks. I also gained an understanding of GPG verification and why validating software sources is important for system security.
