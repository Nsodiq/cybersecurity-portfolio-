## Objective
This lab focused on Linux filesystem interaction, file permissions, and navigating common Linux directories using TryHackMe Linux Fundamentals 2.

## Skills Learned
- Linux filesystem interaction
- File permissions management
- Understanding Linux directories
- User privilege concepts

## Commands Practiced
- ls
- man
- touch
- mkdir
- cp
- mv
- rm
- file
- ls -l
- sudo
- pwd

## Common Directories Learned
- /etc → system configuration files
- /home → user directories
- /root → the root user's home directory
- /var → logs and variable data
- /tmp → temporary files

## File Permissions
Learned how Linux permissions control access to files and directories using ls -l.

## What I Learned
This lab improved my understanding of Linux filesystem interaction, file permissions, and common directories used in Linux environments. These concepts are important for system administration and cybersecurity operations.
