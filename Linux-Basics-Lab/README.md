# Linux Basics Lab

## Objective
Complete Linux Fundamentals 1 on TryHackMe and practice basic Linux commands and navigation.

## Skills Learned
- File navigation
- Linux commands
- File permissions
- Directory management
- Basic terminal usage

## Commands Practiced
- echo
- whoami
- ls
- cd
- cat
- pwd
- find
- wc
- grep

## Shell Operators Learned
- & : run commands in background
- && : run commands sequentially 
- > : redirect output 
- >> : append output

## What I Learned
This lab helped me understand how to navigate Linux systems and use essential terminal commands commonly used in cybersecurity and SOC environments.

## Platform
TryHackMe - Linux Fundamentals 1
